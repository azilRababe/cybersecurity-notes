## Important Analysis Tools

**Process Monitor**
Tracks file, registry, and process activity in real time

**Process Explorer**
Shows process hierarchy, handles, DLLs, and user context

**CFF Explorer**
PE file analysis, hashing, integrity checks

**PEStudio**
Static analysis without executing binaries

**Wireshark**
Detect suspicious traffic, encrypted TLS sessions, exfiltration

**FLOSS**
Extracts and de-obfuscates strings from malware binaries
